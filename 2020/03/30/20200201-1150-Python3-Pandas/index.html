2. 查询特定列的数据

SQL

select flag from xxx;

Pandas

# 2.1 <class 'pandas.core.frame.DataFrame'>
print(type(order_data[['flag', '86_NSN_CM']]))
print(order_data[['flag', '86_NSN_CM']], '\n')
print(type(order_data[['86_NSN_CM']]))

# 2.2 <class 'pandas.core.series.Series'>
print(type(order_data['86_NSN_CM']))

# 3 loc: 按字段定位数据
print(type(order_data.loc[:, ['flag']]))
print(order_data.loc[:, ['flag']])

# 4 iloc: 按索引号索引数据，自0开始
print(type(order_data.iloc[:, [1]]))
print(order_data.iloc[:, [1]])  # 先行再列


# 5 综合
# 列的增删改查：
df["f"] = [1,2,3,4,5]  当f不存在
del df["f"]    df.pop("e")    df.drop("e")
df["f"] = [1,2,3,4,5]  当f存在
df["a"]    df[["a"]]  df[["a", "b"]]   df.loc[:, "a"]

#行的增删改查
df.append(line)
df.drop(["h", "j"])
df.loc["k"] = pd.Series([1,1,1,1,1], index=list("abcde"))
df.loc["a"]   df.iloc[0]

#    行列混合操作
# 先列再行：
df["i"].loc["a"]
# 先行再列：
df.loc["a"]["i"]
# 多行多列：
df[["i", "o", "p"]].loc["b":"d"]
# 获取多行：
df["a":"c"]  df[0:2]   df[[True, False, False]]
# 获取多列：
df[["a", "b"]]

# 复杂的混合操作：
df[df["i"] > 0.5] # 把所有的i列的大于0.5的行，都拿出来！

Tipsinplace不管在那个地方都一样，但是axis也常见，但是意义可能不一样！
Tips直接df[x]先取列，df.loc/df.iloc先取行

numpy

广播运算

Tips不同维度，低维度的列数必须与高纬度的相同，且高维度的行数必须是低维度的整数倍. (2,3,4) — (4)/(3,4)/(2,3,4) 或(2,3,4) — (3,1)

# 原生Python处理数组
list22 = list(range(10))
print(list22)
# 把这个数据序列 0-9 的每个元素都进行 + 1 操作
result_list = []
for i in list22:
    result_list.append(i + 1)
print(result_list)


# numpy处理数组（广播机制）
data = np.arange(10)  # <class 'numpy.ndarray'>
data += 1    # [1,2,3,4] + 1 => [1,2,3,4] + [1,1,1,1] = [2,3,4,5]
display(data)
reshape VS 数组扁平化

reshape维度重构

a = np.arange(1, 13).reshape((2, 3, 4)) # 2堆3行4列 = 三维，会被看作2个两维的，即2个3行4列
b = [10, 20, 30, 40] # 1行1列的b，会被广播为 1个3行4列，再被扩展为2个3行4列
display(a + b)

a0 = np.arange(1, 10)
a1 = a0.reshape((2, -1))  # 用 -1 来自动确定列数
a2 = np.reshape(a0, (2, -1))

扁平化高维变低维. [[1,2], [3,4]] — [1,2,3,4]
ravel返回的是视图，数据引用，影响原值.
flttern返回的是新的浅拷贝，不影响原值.

x = np.arange(1, 13).reshape((3,4), order="C")  # C风格 左到右，上到下
y = np.arange(1, 13).reshape((3,4), order="F")  # F风格 上到下，左到右

# ravel

属性操作

ndim - 维度数
shape - 维度/形状
dtype - 元数数据类型
size - 元素个数
itemsize - 元素长度、字节数
nbytes - 总字节数 = size x itemsize
real - 复数数组的实部数组
imag - 复数数组的虚部数组
T - 数组对象的转置视图
flat - 扁平迭代器

# 构建一个数组
